Categories of Logs:

- User Actions
    1. Search queries
    2. Filter application (e.g., filtering by date, color, etc.)
    3. Pagination (loading more results)
    4. Image selection
    5. Zooming or inspecting an image
    Downloading an image
    Favoriting or saving images
    Interaction with related searches or recommendations
    Uploading an image (if reverse image search is a feature)
- System Actions
    Query results (number of results, time taken)
    Recommendations or suggestions (based on search)
    Error messages (e.g., no results found)
    System performance logs (latency, response time)
    System downtime or crash events
